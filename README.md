# cursor

Stitch-inspired dark mode dashboard for API monitoring.

## Dashboard

See [`dashboard/`](./dashboard/) for the full React application.

```bash
cd dashboard
npm install
npm run dev
```

Open http://localhost:5173 to view the dashboard.

## Design

The dashboard follows [Google Stitch](https://stitch.withgoogle.com) design patterns with:

- Dark mode theme with emerald accents
- Glass-morphism cards and sidebar
- KPI metrics, latency charts, and API status table
- Full design system in [`dashboard/DESIGN.md`](./dashboard/DESIGN.md)
