Azerbaycan Stratejik Araştırmalar Derneği (ASAD) web sitesi — Stitch tasarımlarından dönüştürülmüş statik HTML portal.

## Canlı Site

**GitHub Pages:** https://eyzenent.github.io/cursor/

Tüm sayfalar: https://eyzenent.github.io/cursor/sayfalar.html

## GitHub Pages Kurulumu (tek seferlik)

Repo public ve `gh-pages` branch hazır. Yayına almak için:

1. [Repository Settings → Pages](https://github.com/eyzenent/cursor/settings/pages) sayfasına gidin
2. **Build and deployment → Source:** `Deploy from a branch`
3. **Branch:** `gh-pages` → klasör: `/ (root)`
4. **Save** — birkaç dakika içinde site yayında olur
