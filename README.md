# andante

Figma tasarımından üretilmiş Flutter mobil uygulaması.

## Ekranlar

- **Ana Akış** — gönderi akışı, alıntı kartları ve FAB
- **Keşfet** — arama, trend konular, önerilen içerik üreticileri ve keşif grid'i
- **Bildirimler** — mention, beğeni, takip ve sistem bildirimleri
- **Profil** — kullanıcı profili ve son kürasyonlar

## Kurulum

```bash
cd andante_app
flutter pub get
flutter run
```

## APK

Release APK dosyası:

`andante_app/build/app/outputs/flutter-apk/app-release.apk`

APK'yı yeniden oluşturmak için:

```bash
cd andante_app
flutter build apk --release
```
