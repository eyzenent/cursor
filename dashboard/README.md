# Nexus Dashboard

Stitch-inspired dark mode API monitoring dashboard built with React, Tailwind CSS, and Recharts.

## Features

- Dark mode UI with glass-morphism effects
- KPI metric cards with trend indicators
- Real-time latency chart
- API endpoints table with status badges
- Activity feed sidebar
- Fully responsive layout

## Quick Start

```bash
npm install
npm run dev
```
