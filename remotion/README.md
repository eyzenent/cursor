# Remotion (ASAD)

Programatik video: Instagram Reels formatında belgesel tarzı ETF anlatımı.

## Studio

```bash
cd remotion
npm install
npm run dev
```

Composition: **EtfReels** — 1080×1920, 60 sn, müzikli belgesel.

## Render

```bash
npm run render
# → out/etf-reels.mp4
```

## Müzik
